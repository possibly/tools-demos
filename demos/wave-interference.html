Four-Corner Wave Interference

Click the button to see waves spawning from all four corners and colliding! Watch as multiple waves create complex interference patterns that ripple through the button's edges with many peaks and valleys, demonstrating advanced wave superposition and how waves naturally fizzle out after collision.

Four-Corner Wave Interference Physics

Wave Superposition: ψ_total = ψ₁ + ψ₂ + ψ₃ + ψ₄
Wave Equation: ψ(x,y,t) = A·sin(k·r - ωt + φ)
Superposition: Constructive + Destructive interference
Damping: Amplitude decreases as e^(-γt)
A = amplitude, k = wave number
ω = angular frequency, φ = phase
r = distance from source
